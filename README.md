# 🐍 Python Cybersecurity & OSINT Tools 🛡️

Class 10 practical implementation tools for target parsing, banner generation, and dynamic OSINT reconnaissance.

## 🚀 Tools Included:
1. **Target Parser & Helper (`target_helper.py`)**: Standardizes input domains, counts length, and structures URL protocols.
2. **OSINT Target Recon (`cyber_osint.py`)**: Traces target IPs to extract geographic locations, ISP ownership, and server network profiles.
